## Usage
```
# Clone this repository
$ git clone https://github.com/philipkondrenko/Note-App

# cd into the repository
$ cd Note-App

# cd into the frontend folder
$ cd frontend

# Install dependencies
$ npm install

# Run React server
$ npm start

```
